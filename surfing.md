```
 # 电脑主机组成部分：
- CPU：中央处理器：负责执行程序命令
	主频：工作效率或运算速度，主频越高，在单位时间内的指令数据就越多，提升处理速度。游戏向选择主频高的。
	核心：工作独立处理单元数量，核心越多，可以同时完成的任务就越多。
	线程：一个核心能够处理的独立任务流。双核四线程，说明一个核有两个线程。图形渲染，需要多核心多线程。
	架构：每一代的架构都是一样的，最新一代往往代表最高生产力，办公选择前一到两代。
- 主板：所有硬件组件连接的平台，提供数据传输的通道
- RAM：内存：临时存储数据，以便CPU快速访问
- 存储设备：固态硬盘（速度快，价格高），机械硬盘（速度慢，空间较大）
- GPU：显卡：处理图形和视频渲染，队友游戏和图形设计比较重要
	英伟达 & AMD
- 电源：
- 机箱：
- 散热器：风扇之类，散热比较重要。
```



## 游戏

```
 # Game
- 上古卷轴V：经典中的经典：2011年11月11日发售。年度最佳游戏。绝伦和烽火的整合包质量较高，玉强整合稳定性较差，且吃配置，暂不考虑。
- 艾尔登法环：2022年发售，同样是年度最佳。中世纪剑与魔法的风格，乔治马丁参与构建世界背景。但是由于魂类游戏的风格，挑战性很大。因为上班，不可能抽出太多时间，单人游玩需要开挂才能体会到游戏的魅力。或者考虑和冰冰一起玩，双人可以极大降低游戏难度，且有一路陪伴的感觉。但是冰冰晕3D，如果有两台机器之后可以考虑。
- 黑暗之魂：纯魂类游戏，剧情悲壮。骑士精神，剑与盾，难度适中，适合新手入坑，目前考虑全收集。
- 双人成行：年度最佳，双人游戏首选，剧情，画面无可挑剔。考虑到冰冰一块玩需要花费时间较长，需慢慢引导。
- 胡闹厨房：又名分手厨房，双人配合。精气神充足的时候才能考虑。
```
